List of repositories to patch:

  1. android_build
  2. android_frameworks_av
  3. android_frameworks_base
  4. android_vendor_pa

android_build

  • removing auditd from compilation
  • removing webviewchromium from bootclasspath
  • revert adding prebuilt chromium support scripts

android_frameworks_av

  • Camera: QCOM legacy definitions,HTC cam switch,NO_UPDATE_PREVIEW
  • Added support legacy Qualcomm OpenMAX IL libs for MSM7x27
  • libstagefright: Enable meta mode for 7x27a video encoder
  • legacy OMX: handle QCOM colorformat
  • use hw_get_module_by_class
  • Remove CAMERA_CMD_METADATA && LONGSHOT calls

android_frameworks_base

  • add resized missing drawables to drawable-mdpi
  • fix for metadata, longshot

android_vendor_pa

  • products/: add pico defines
